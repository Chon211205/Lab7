## Descripción
Implementación del juego clásico Snake utilizando React. El jugador controla una serpiente que se mueve dentro de un tablero, cuyo objetivo es comer la comida para crecer y aumentar su puntaje. El juego termina si la serpiente choca contra los bordes o contra sí misma. A medida que avanza, la velocidad aumenta, haciendo el juego más desafiante.

## Controles
- Flechas del teclado:
  - ⬆️ Arriba  
  - ⬇️ Abajo  
  - ⬅️ Izquierda  
  - ➡️ Derecha  

- Enter:
  - Iniciar el juego  
  - Reiniciar después de perder
